# 贪心算法：构造点染色数 ≤ Δ+1

## 算法简介

**问题**：给定无向图 $G=(V,E)$，给顶点染色使相邻顶点颜色不同，保证颜色数 ≤ Δ+1。

这是 Brooks 定理弱化版的构造性证明——贪心总能找到 ≤ Δ+1 的合法点染色。

## 算法流程

1. 将顶点按任意顺序排列 $v_1, v_2, \ldots, v_n$
2. 对每个顶点 $v_i$：
   - 查看所有**已染色**邻居用了哪些颜色
   - 从 $\{1,2,\ldots,\Delta+1\}$ 中选**最小**未用颜色
3. 染上该颜色

**正确性**：每个顶点最多 Δ 个邻居，最坏用掉 Δ 种，Δ+1 中总有可用的。

**Welsh-Powell 优化**：按度数降序排列后再贪心，通常效果更好。

## 手动模拟示例

### 三角形 K3 (Δ=2)

```
    2
   /\
  /  \
 /    \
0──────1
```

顺序 0→1→2：

| 步骤 | 顶点 | 邻居已用色 | 可选色 | 选择 |
|------|------|-----------|--------|------|
| 1 | 0 | (无) | {1,2,3} | **1** |
| 2 | 1 | {1} | {2,3} | **2** |
| 3 | 2 | {1,2} | {3} | **3** |

结果: color=[1,2,3]，3 色，恰好最优（奇环需 3 色）。

### 四边形 C4

```
0───1
|   |
3───2
```

顺序 0→1→2→3：

| 步骤 | 顶点 | 邻居已用 | 选择 |
|------|------|---------|------|
| 1 | 0 | - | **1** |
| 2 | 1 | 0=1 | **2** |
| 3 | 2 | 1=2 | **1** |
| 4 | 3 | 0=1,2=1 | **2** |

结果只用 2 色，最优（偶环 χ=2）。

## 时间复杂度

- 每顶点检查 ≤Δ 个邻居: $O(n \cdot \Delta) = O(m)$
- Welsh-Powell 排序: $O(n \log n)$
- 总: $O(n \log n + m)$
