# 二分图正常 Δ 边染色分治算法

## 算法简介

**问题**：给定二分图，用 Δ 种颜色对其边进行正常染色（相邻边不同色）。

**定理 (König)**：二分图的边色数 = Δ。

**算法**：用分治策略，递归地将图分解为两个子图，各自染色后再合并。

## 算法流程

1. 若图的最大度 Δ = 0，返回空染色
2. 若 Δ 为偶数：
   - 将图分解为两个欧拉子图（用 EulerPartition 将边分为两组，每组 Δ/2 正则）
   - 递归对各子图染色
3. 若 Δ 为奇数：
   - 用 MDMatching 算法找 Δ-正则图的完美匹配
   - 给匹配边染色 Δ
   - 删除匹配边，剩余图最大度 Δ-1，递归处理

### 子算法

**EulerPartition**：将偶数度正则二分图分解为两个等度子图。
- 对每个连通分量求欧拉回路
- 交替着两种"颜色"（对应两个子图）

**MDMatching**：在 Δ-正则二分图中找完美匹配。
- 可用匈牙利算法（每轮找增广路）

## 手动模拟示例

### 完全二分图 K2,2 (Δ=2)

```
L0 ── R0
  \  /
   \/
   /\
  /  \
L1 ── R1
```

边: (L0,R0)(L0,R1)(L1,R0)(L1,R1), Δ=2

**第 1 层 (Δ=2, 偶数)**：
- EulerPartition: 将边分为两组，每组 2 条边
  - 组1: (L0,R0), (L1,R1)  (一个完美匹配)
  - 组2: (L0,R1), (L1,R0)  (另一个完美匹配)
- 递归处理组1 (Δ=1): 染色 1
- 递归处理组2 (Δ=1): 染色 2

**结果**：
- (L0,R0)=1, (L1,R1)=1
- (L0,R1)=2, (L1,R0)=2

用 2 = Δ 种颜色完成染色 ✓

### 完全二分图 K3,3 (Δ=3)

**第 1 层 (Δ=3, 奇数)**：
- MDMatching 找完美匹配: 例如 (L0,R0)(L1,R1)(L2,R2)
- 给匹配边染色 3
- 删除后剩余图 Δ=2

**第 2 层 (Δ=2, 偶数)**：
- 分解后递归染色 1 和 2

**结果**：用 3 = Δ 种颜色完成染色

## 时间复杂度

- EulerPartition: $O(E)$
- MDMatching (匈牙利): $O(VE)$
- 递归深度: $O(\log \Delta)$
- 总: $O(VE \log \Delta)$

## 测试用例

1. K2,2 (Δ=2)：用 2 色
2. 四边形 (Δ=2)：用 2 色
3. 链式二分图：验证 Δ 色
4. K3,3 (Δ=3)：用 3 色
