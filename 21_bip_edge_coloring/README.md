# 二分图正常 Δ 边染色算法

## 算法简介

**问题**：给定二分图，用 Δ 种颜色对其边进行正常染色（相邻边不同色）。

**定理 (König)**：二分图的边色数 = Δ。

**本实现**：使用贪心染色 + 交替路径颜色翻转的方法。对每条边 $(u,v)$，尝试找到 $u$ 和 $v$ 处都空闲的最小颜色；若无共同空闲颜色，则通过交替路径翻转来释放颜色。

> **理论算法**（分治）：若 Δ 为偶数，用 EulerPartition 分成两个 Δ/2 子图递归；若 Δ 为奇数，先找完美匹配染 Δ 色，剩余递归。本实现采用等价但更简洁的贪心+翻转方法。

## 算法流程

1. 计算图的最大度 Δ
2. 对每条边 $(u,v)$ 依次处理：
   a. 找 $u$ 和 $v$ 处都空闲的最小颜色 $c$（从 1 开始）
   b. 若找到 → 直接染 $c$
   c. 若未找到（所有 Δ 种颜色在 $u$ 或 $v$ 处都被占用）：
      - 找 $u$ 空闲的颜色 $a$，$v$ 空闲的颜色 $b$
      - 从 $v$ 出发沿 $a$-$b$ 交替路径翻转颜色
      - 翻转后 $a$ 在 $v$ 处空闲，染上 $a$
3. 验证染色合法性（相邻边不同色，颜色数 ≤ Δ）

## 手动模拟示例

### 完全二分图 K2,2 (Δ=2)

```
L0 ── R0
  \  /
   \/
   /\
  /  \
L1 ── R1
```

边: (L0,R0)(L0,R1)(L1,R0)(L1,R1), Δ=2

| 步骤 | 处理边 | 动作 | 染色结果 |
|------|--------|------|---------|
| 1 | (L0,R0) | L0空闲={1,2}, R0空闲={1,2}, c=1可用 | color=1 |
| 2 | (L0,R1) | L0已用{1}, 空闲{2}, R1空闲={1,2}, c=2可用 | color=2 |
| 3 | (L1,R0) | L1空闲={1,2}, R0已用{1}, 空闲{2}, c=2可用 | color=2 |
| 4 | (L1,R1) | L1已用{2}, R1已用{2}, c=1在L1空闲但R1不可用... | 交替路径翻转 |

最终用 2 = Δ 种颜色完成染色 ✓

## 时间复杂度

- 每条边处理：$O(\Delta + n)$
- 交替路径翻转：$O(n)$
- 总：$O(m \cdot n)$

## 测试用例

1. K2,2 (Δ=2)：用 2 色
2. 四边形 (Δ=2)：用 2 色
3. 链式二分图 (Δ=3)：用 3 色
4. K3,3 (Δ=3)：用 3 色
5. K4,4 (Δ=4)：用 4 色
