# 埃德蒙兹-约翰逊：中国邮递员问题

## 算法简介

**中国邮递员问题 (Chinese Postman Problem)**：给定无向连通图，每条边有权重，找一条经过每条边至少一次且总权最小的回路（允许重复经过边）。

**核心思想**：若图是欧拉图（所有顶点度数为偶），则只需走一次欧拉回路即可。否则，需要"重复走"某些边来配对奇度顶点。

## 算法流程

1. 统计所有顶点度数，找出**奇度顶点**（必有偶数个）
2. 对所有奇度顶点求**最小权完美匹配**（两两配对）
3. 配对的两顶点之间沿最短路径添加"重复边"
4. 结果图所有顶点度数为偶 → 欧拉图
5. 求欧拉回路，总权 = 原边权和 + 重复边权和

## 手动模拟示例

### 图结构

```
0───1───2
```

边权: (0,1)=5, (1,2)=3

奇度顶点: 0(deg=1), 2(deg=1)

### 匹配

奇度顶点对: {0, 2}，最短路径距离 = 5+3 = 8

### 结果

- 原边权和: 5+3 = 8
- 重复边权: 8（从 0 走到 2 再走回 0 的代价）
- 最优邮路总权: 16

实际路线: 0→1→2→1→0（经过边 0-1 两次，1-2 两次）

### 另一个例子

```
0───1
|＼  |
| ＼ |
3───2
```

（正方形+对角线，边权全为 1）

| 顶点 | 度数 | 奇度? |
|------|------|-------|
| 0 | 3 | 是 |
| 1 | 2 | 否 |
| 2 | 3 | 是 |
| 3 | 2 | 否 |

奇度: {0,2}，最短路径 0-2 = min(1, 1+1=2) = 1

- 原边权和: 5
- 重复边权: 1
- 最优邮路: 6

## 时间复杂度

- Floyd-Warshall: $O(V^3)$
- DP 最小权匹配: $O(k^2 \cdot 2^k)$, $k$ 为奇度顶点数
- 总: $O(V^3 + k^2 \cdot 2^k)$

## 测试用例

1. 正方形（无奇度）：总权=10
2. 三角形（无奇度）：总权=6
3. 路径（2 奇度）：总权=16（需重复走）
4. 正方形+对角线（2 奇度）：总权=18
