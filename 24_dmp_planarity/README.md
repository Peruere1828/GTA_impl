# DMP 可平面性算法

## 算法简介

**DMP 算法**（Demoucron-Malgrange-Pertuiset, 1964）是判断一个图是否为平面图的经典算法。

**问题**：给定无向图，判断它是否可以画在平面上使得边不相交（除端点外）。

**Kuratowski 定理**：图是平面图 ⇔ 不含 K5 或 K3,3 的细分。

DMP 算法不直接检测 Kuratowski 子图，而是通过**增量构造平面嵌入**的方法。本实现包含以下组件：

1. **Euler 公式 + 围长边界**：快速排除明显非平面图
2. **边界面分裂**：用边界顶点序列表示面，嵌入段时正确分裂面
3. **增量嵌入**：将未嵌入边分组为段，逐段嵌入到面中

## 算法流程

1. 在图中找一个**基本圈**（cycle）
2. 基本圈将平面分为**内部**和**外部**两个面（边界序列方向相反）
3. 将剩余边分为**段 (segment)**：
   - 一段 = 未嵌入边的连通分量
   - 段与已嵌入部分的共享顶点称为**接触点**
4. 对每个段，找出所有接触点都位于其边界上的面
5. 将段嵌入到某个有效面中：
   - **0 接触点**：独立连通分量，直接嵌入
   - **1 接触点**：附着在边界上的树，在接触点处展开边界
   - **2 接触点**：连接边界上两点，沿段内路径将面**分裂为两个新面**
   - **≥3 接触点**：复杂情况，简化处理
6. 重复 3-5 直到：
   - 所有边都嵌入 → 图是平面图
   - 某段无处可嵌 → 图不是平面图

## 关键数据结构

- **面 (Face)**：由边界上的边序列定义
- **段 (Segment)**：待嵌入的边或路径

## 手动模拟示例

### K4 (是平面图)

```
   0
  /|\
 / | \
1--2--3
 \ | /
  \|/
   4 (不在K4中)
```

边: (0,1)(0,2)(0,3)(1,2)(2,3)(1,3)

**步骤 1**：找基本圈 0-1-2-0
- 面: 内部 = {0→1→2→0}, 外部 = {0→2→1→0}

**步骤 2**：分段
- 段 1: 边 (0,3) — 端点 {0,3}, 面: 内部 ✓, 外部 ✓
- 段 2: 边 (1,3) — 端点 {1,3}, 面: 内部 ✓, 外部 ✓
- 段 3: 边 (2,3) — 端点 {2,3}, 面: 内部 ✓, 外部 ✓

**步骤 3**：选择一个段嵌入（如 (0,3) 嵌入内部）
- 内部面分裂为两个: {0→3→2→0} 和 {0→1→2→3→0}

**继续**：最终所有边都能嵌入 → K4 是平面图 ✓

### K3,3 (不是平面图)

```
L0──R0
| ＼/ |
| /＼ |
L1──R1
| ＼/ |
| /＼ |
L2──R2
```

边: (L0,R0)(L0,R1)(L0,R2)(L1,R0)(L1,R1)(L1,R2)(L2,R0)(L2,R1)(L2,R2)

DMP 会在某一步发现某段无处可嵌 → 图不是平面图 ✗

## 时间复杂度

- 每次迭代需扫描所有段和面：$O(n^2)$
- 总迭代次数：$O(n)$
- 总复杂度：$O(n^3)$

## 测试用例

### 通用 DMP (dmpPlanarity)

1. K4：平面图 ✓
2. K5：非平面图 ✓（Euler 公式: 10 > 3×5-6=9）
3. K3,3：非平面图 ✓（girth=4: 9 > 2×6-4=8）
4. 正方形 C4：平面图 ✓
5. 树：平面图 ✓
6. Petersen 图：非平面图 ✓（girth=5: 15 > 5×(10-2)/3≈13）
7. 正方形+对角线：平面图 ✓（需正确分裂面）
8. K4 变体（正方形+两条对角线）：平面图 ✓
9. 5 点三角剖分：平面图 ✓
10. 正八面体骨架（6 顶点 12 边）：平面图 ✓

### 2-连通特化版 (dmpPlanarity2Connected)

**简化点**：无需处理 0/1 接触点，面边界必为简单环。≥3 接触点时沿面边界顺序正确分裂。

1. K4：平面图 ✓
2. K5：非平面图 ✓
3. K3,3：非平面图 ✓
4. C4：平面图 ✓
5. Petersen：非平面图 ✓
6. 正方形+对角线：平面图 ✓（2 接触点面分裂）
7. 正八面体：平面图 ✓
8. 轮图 W5：平面图 ✓（5 接触点分裂）
9. 5 点三角剖分：平面图 ✓
10. 正十二面体骨架（20 顶点 30 边）：平面图 ✓
