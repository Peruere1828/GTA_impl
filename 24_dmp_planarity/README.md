# DMP 可平面性算法

## 算法简介

**DMP 算法**（Demoucron-Malgrange-Pertuiset, 1964）是判断一个图是否为平面图的经典算法。

**问题**：给定无向图，判断它是否可以画在平面上使得边不相交（除端点外）。

**Kuratowski 定理**：图是平面图 ⇔ 不含 K5 或 K3,3 的细分。

DMP 算法不直接检测 Kuratowski 子图，而是通过**增量构造平面嵌入**的方法。本实现为简化版本，结合 Euler 公式和围长(girth)边界做快速排除，再通过增量嵌入做最终判定。

## 算法流程

1. 在图中找一个**基本圈**（cycle）
2. 基本圈将平面分为**内部**和**外部**两个面
3. 将剩余边分为**段 (segment)**：
   - 一段 = 一条不在已嵌入部分的边，其端点都在已嵌入部分
   - 或一条从已嵌入顶点出发的未嵌入路径
4. 对每个段，找出它可以嵌入的**面集合**（即其所有端点都在的面）
5. 选择可嵌入面数**最少**的段（优先嵌入受限段）
6. 将选中的段嵌入到某个可用的面中
7. 重复 3-6 直到：
   - 所有边都嵌入 → 图是平面图
   - 某段无处可嵌 → 图不是平面图

## 关键数据结构

- **面 (Face)**：由边界上的边序列定义
- **段 (Segment)**：待嵌入的边或路径

## 手动模拟示例

### K4 (是平面图)

```
   0
  /|\
 / | \
1--2--3
 \ | /
  \|/
   4 (不在K4中)
```

边: (0,1)(0,2)(0,3)(1,2)(2,3)(1,3)

**步骤 1**：找基本圈 0-1-2-0
- 面: 内部 = {0→1→2→0}, 外部 = {0→2→1→0}

**步骤 2**：分段
- 段 1: 边 (0,3) — 端点 {0,3}, 面: 内部 ✓, 外部 ✓
- 段 2: 边 (1,3) — 端点 {1,3}, 面: 内部 ✓, 外部 ✓
- 段 3: 边 (2,3) — 端点 {2,3}, 面: 内部 ✓, 外部 ✓

**步骤 3**：选择一个段嵌入（如 (0,3) 嵌入内部）
- 内部面分裂为两个: {0→3→2→0} 和 {0→1→2→3→0}

**继续**：最终所有边都能嵌入 → K4 是平面图 ✓

### K3,3 (不是平面图)

```
L0──R0
| ＼/ |
| /＼ |
L1──R1
| ＼/ |
| /＼ |
L2──R2
```

边: (L0,R0)(L0,R1)(L0,R2)(L1,R0)(L1,R1)(L1,R2)(L2,R0)(L2,R1)(L2,R2)

DMP 会在某一步发现某段无处可嵌 → 图不是平面图 ✗

## 时间复杂度

- 每次迭代需扫描所有段和面：$O(n^2)$
- 总迭代次数：$O(n)$
- 总复杂度：$O(n^3)$

## 测试用例

1. K4：平面图 ✓
2. K5：非平面图 ✓（Euler 公式: 10 > 3*5-6=9）
3. K3,3：非平面图 ✓
4. 正方形：平面图 ✓
5. Petersen 图：非平面图 ✓
