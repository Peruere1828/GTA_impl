# 有向图强连通分支 Tarjan 算法

## 算法简介

**强连通分量（SCC）**：有向图的极大子图，其中任意两点互相可达。

**Tarjan 算法**：一次 DFS 即可找出所有 SCC，复杂度 $O(V+E)$。

## 核心概念

- `dfn[u]`：DFS 访问顺序编号
- `low[u]`：$u$ 及其后代通过后向边/横叉边能到达的**仍在栈中**的最早顶点 dfn
- `inStack[u]`：顶点 $u$ 是否在当前 DFS 栈中

## 算法流程

1. 对每个未访问顶点做 DFS
2. 进入 $u$ 时：`dfn[u]=low[u]=++timer`，入栈
3. 对 $u$ 的每条出边 $(u,v)$：
   - 若 $v$ 未访问：递归访问，`low[u]=min(low[u], low[v])`
   - 若 $v$ 在栈中：`low[u]=min(low[u], dfn[v])`
4. 若 `low[u] == dfn[u]`：从栈顶弹出直到 $u$，这些顶点构成一个 SCC

**关键判断**：`low[u] == dfn[u]` 表示 $u$ 是其所在 SCC 中最早被访问的顶点（SCC 的"根"）。

## 手动模拟示例

### 有向图

```
0 → 1 → 2 → 3
    ↑         ↓
    └─────────┘  (还有 4→5→6→4 的环)
```

边: 0→1, 1→2, 2→3, 3→1, 4→5, 5→6, 6→4

### DFS 过程

| 步骤 | 进入 | dfn | low | 栈 | SCC? |
|------|------|-----|-----|-----|------|
| 1 | 0 | 1 | 1 | [0] | |
| 2 | 0→1 | 1→2 | | [0,1] | |
| 3 | 1→2 | 2→3 | | [0,1,2] | |
| 4 | 2→3 | 3→4 | | [0,1,2,3] | |
| 5 | 3→1 | | | | low[3]=min(4,2)=2 |
| 6 | 回溯3 | | low[3]=2 | | low[3]=2 ≠ dfn[3]=4 |
| 7 | 回溯2 | | low[2]=min(3,2)=2 | | low[2]=2 ≠ dfn[2]=3 |
| 8 | 回溯1 | | low[1]=min(2,2)=2 | | low[1]=2 = dfn[1]=2! |
| | | | | | 弹出3,2,1 → SCC: {1,2,3} |
| 9 | 回溯0 | | low[0]=min(1,2)=1 | [0] | low[0]=1=dfn[0]=1! |
| | | | | | 弹出0 → SCC: {0} |

同理处理另一连通分量 4,5,6 → SCC: {4,5,6}

**结果**：SCC1={1,2,3}, SCC2={0}, SCC3={4,5,6}

## 时间复杂度

- DFS 一次遍历：$O(V+E)$
- 每个顶点进出栈一次

## 测试用例

1. 3-环：1 个 SCC
2. 两条链：5 个 SCC（每个点一个）
3. 两个 SCC 通过边连接（实际合并为一个）
4. CLRS 经典例子
5. DAG：每个点一个 SCC
