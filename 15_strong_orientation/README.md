# 强连通定向算法

## 算法简介

**问题**：给定一个无割边的无向连通图 $G=(V,E)$（2-边连通图），对每条边赋予方向，使结果有向图是强连通的。

该算法是 **Robbins 定理**（1939）的构造性证明：无向图能被定向为强连通有向图 ⇔ 它是 2-边连通的。

## 算法流程

1. 从任意顶点 $s$ 出发，对无向图做 DFS，得到 DFS 树
2. 边分为两类定向：
   - **树边**：父节点 → 子节点（从根向外）
   - **后向边**（非树边）：后代 → 祖先（向根方向）
3. 输出所有有向边

### 正确性直觉

- 沿树边从根可到所有节点
- 沿树边向下到叶，再通过后向边向上跳，最终可回到根
- 任意两点 $u→v$：u→根→v

## 手动模拟示例

### 图结构（正方形+对角线，2-边连通）

```
    0 ──── 1
    | \     |
    |  \    |
    3 ──── 2
```

n=4, 边集: (0,1)(0,2)(0,3)(1,2)(2,3)

> 验证 2-边连通：删除任意一条边后图仍连通，无割边。

### DFS 过程（从 0 出发，邻接表顺序: 1, 2, 3）

| 步骤 | 当前 | 动作 |
|------|------|------|
| 1 | 0 | 进入0, vis[0]=1 |
| 2 | 0→1 | 树边 0→1, 递归进1 |
| 3 | 1→2 | 树边 1→2, 递归进2 |
| 4 | 2→0 | 后向边 2→0 (0已访问, 是祖先) |
| 5 | 2→3 | 树边 2→3, 递归进3 |
| 6 | 3→0 | 后向边 3→0 (0已访问, 是祖先) |

### 定向结果

```
树边:       0→1, 1→2, 2→3
后向边:     2→0, 3→0
```

### 验证强连通

- 从 0 可沿树边到达任意节点: 0→1→2→3 ✓
- 从 1→0: 1→2→0 ✓
- 从 2→1: 2→0→1 ✓
- 从 3→1: 3→0→1 ✓
- 任意两点互相可达 → 强连通 ✓

## 时间复杂度

- DFS: $O(V+E)$，每条边处理一次

## 注意事项

- 图必须无割边（2-边连通），否则算法无效
- DFS 起始点和邻接表顺序影响具体定向结果，但不影响强连通性
