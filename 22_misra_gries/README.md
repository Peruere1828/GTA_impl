# 米什拉-格莱斯 (Misra-Gries) 算法：边染色数 ≤ Δ+1

## 算法简介

**问题**：给无向图的边染色，相邻边（共享顶点）不同色，使用 ≤ Δ+1 种颜色。

这是 **Vizing 定理**的构造性证明：任何简单图的边色数 ∈ {Δ, Δ+1}。

## 算法流程

对每条边 $(u,v)$ 依次染色：

1. 找 $u$ 和 $v$ 处都空闲的最小颜色 $c$
2. 若 $c$ 存在 → 直接染 $c$
3. 否则：
   - 找 $u$ 空闲的颜色 $a$，$v$ 空闲的颜色 $b$
   - 构建从 $v$ 出发的 $a$-$b$ 交替路径
   - 沿路径翻转颜色（$a$→$b$, $b$→$a$）
   - 翻转后 $a$ 在 $v$ 处空闲，染上 $a$

**关键概念**：
- **Vizing 扇**：以 $u$ 为中心的边序列
- **交替路径**：边颜色交替为 $a$ 和 $b$ 的极大路径

## 手动模拟示例

### 三角形 K3 (Δ=2)

边: (0,1), (1,2), (2,0)

| 步骤 | 处理边 | 动作 | 染色结果 |
|------|--------|------|---------|
| 1 | (0,1) | 0和1都空闲色1 → 直接染1 | color(0,1)=1 |
| 2 | (1,2) | 1空闲={2,3}, 2空闲={1,3} | c=1: 1处被用, 跳过 |
| | | c=2: 1处空闲, 2处空闲 → 染2 | color(1,2)=2 |
| 3 | (2,0) | 2空闲={1,3}, 0空闲={2,3} | c=1: 0处被用, 跳过 |
| | | c=2: 2处被用, 跳过 | |
| | | c=3: 2处空闲, 0处空闲 → 染3 | color(2,0)=3 |

最终: 边(0,1)=1, (1,2)=2, (2,0)=3, 使用 3=Δ+1 色 ✓

> 注：K3 是奇环，Δ=2，但二染色边需要 3 色（Vizing 定理给出上界 Δ+1=3）。

## 时间复杂度

- 每条边处理：$O(n)$ 找空闲颜色 + $O(n)$ 交替路径
- 总：$O(nm)$

## 测试用例

1. 三角形 K3：Δ=2, 用 3 色
2. 四边形 C4：Δ=2, 用 2 色
3. 星形 K1,4：Δ=4, 用 4 色
4. Petersen 图：Δ=3, 用 ≤4 色
5. 完全图 K4：Δ=3, 用 ≤4 色
