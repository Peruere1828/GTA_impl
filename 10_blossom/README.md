# 埃德蒙兹花算法 (Blossom Algorithm)：一般图最大匹配

## 算法简介

**问题**：在一般图（非二分图）中求最大匹配。

**花算法**由 Jack Edmonds (1965) 提出，是第一个多项式时间的一般图最大匹配算法。复杂度 $O(V^3)$。Micali-Vazirani 可优化到 $O(E\sqrt{V})$。

## 核心概念

### 花 (Blossom)

长度为奇数的交替环（奇数条边，其中匹配边比非匹配边多 1 条或少 1 条）。

### 缩花 (Contracting)

将花收缩为一个"超级顶点"，在缩点后的图中继续找增广路。

**关键性质**：原图中有增广路当且仅当缩花后的图中有增广路。

## 算法流程

1. 初始化空匹配
2. 对每个未匹配顶点尝试找增广路：
   - **BFS/DFS** 搜索交替森林
   - 若遇到已匹配顶点：扩展交替路径
   - 若遇到同类型顶点：发现花 → **缩花**
   - 若遇到未匹配顶点：找到增广路 → **扩路**
3. 找到增广路后翻转匹配状态
4. 重复直到无增广路

### 状态标记

- 偶数层顶点（Even/Outer）：从根经过偶数条边到达 → 可扩展
- 奇数层顶点（Odd/Inner）：从根经过奇数条边到达
- 未标记顶点

### 发现花

若 BFS 发现一条边连接两个偶数层顶点，则形成一个奇环（花）。
- 找到两个顶点的最近公共祖先
- 环上的所有顶点收缩为一个超级顶点

### 扩路

若遇到未标记顶点且它未匹配 → 找到增广路。
- 沿 parent 指针回溯，翻转路径上所有边的匹配状态

## 手动模拟示例

### 五边形 (奇环)

```
   0───1
  /     \
 4       2
  \     /
   3───┘
```

边: (0,1)(1,2)(2,3)(3,4)(4,0)

**步骤 1**：当前匹配为空。
- BFS 从 0 出发
- 0(Even) → 1(Odd)
- 1 → 2(Even)  
- 2 → 3(Odd)
- 3 → 4(Even)
- 4 → 0(Even) ← **花!** 0 和 4 都是偶数层!

**步骤 2**：缩花。整个五边形收缩为一个超级顶点。
- 在缩点后的图中找增广路（可能没有）
- 展开花，花内可能有内部增广路

**步骤 3**：花内部: 0-4 之间的奇数路径, 翻转后得到匹配边 (0,1) 和 (3,4), 匹配数=2

（匹配大小最大为 2，因为 5 个顶点奇数个）

## 时间复杂度

- Edmonds 原始: $O(V^3)$
- Gabow: $O(V^3)$
- Micali-Vazirani: $O(E\sqrt{V})$

## 测试用例

1. 三角形：匹配数=1
2. 五边形：匹配数=2
3. 包含花的图：验证正确处理
4. 二分图（与匈牙利算法一致）：验证一般图兼容
