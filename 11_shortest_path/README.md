# Dijkstra 与 Bellman-Ford 最短路算法

## 算法简介

### Dijkstra 算法

求解**非负权**有向图（或无向图）的单源最短路径。使用优先队列优化，复杂度 $O((V+E)\log V)$。

**核心思想**：贪心——每次从"未确定最短距离"的顶点中选距离最小的，松弛其邻边。

### Bellman-Ford 算法

可处理**含负权边**的图，并且能**检测负权环**。复杂度 $O(VE)$。

**核心思想**：对所有边反复松弛 $V-1$ 轮。若第 $V$ 轮仍能松弛，则存在负环。

## Dijkstra 算法流程

1. 初始化 `dist[src]=0`，其他为 $\infty$；所有顶点入优先队列
2. 每次取出 `dist` 最小的顶点 $u$
3. 对 $u$ 的每条出边 $(u,v,w)$：
   - 若 `dist[v] > dist[u] + w`，则更新并放入优先队列
4. 重复 2-3 直到队列为空

## Bellman-Ford 算法流程

1. 初始化 `dist[src]=0`，其他为 $\infty$
2. 重复 $V-1$ 轮：对每条边松弛
3. 再做一轮：若仍有边可松弛 → 存在负权环

## 手动模拟示例

### Dijkstra

图：0→1:10, 0→2:3, 1→2:1, 1→3:2, 2→3:8, 2→4:2, 3→4:7

求源点 0 到各点最短距离。

| 轮次 | 取出 | dist 数组 | 松弛操作 |
|------|------|-----------|---------|
| 初始 | - | [0, ∞, ∞, ∞, ∞] | - |
| 1 | 0 | [0, 10, 3, ∞, ∞] | dist[1]=10, dist[2]=3 |
| 2 | 2 | [0, 4, 3, 11, 5] | dist[1]=min(10,3+1)=4, dist[3]=11, dist[4]=5 |
| 3 | 1 | [0, 4, 3, 6, 5] | dist[3]=min(11,4+2)=6 |
| 4 | 4 | [0, 4, 3, 6, 5] | dist[3]=min(6,5+7)=6(不变) |
| 5 | 3 | [0, 4, 3, 6, 5] | (无更新) |

**最终**: dist = [0, 4, 3, 6, 5]

### Bellman-Ford（含负权边）

边: 0→1:-1, 0→2:4, 1→2:3, 1→3:2, 1→4:2, 3→2:5, 3→1:1, 4→3:-3

| 轮 | dist[0] | dist[1] | dist[2] | dist[3] | dist[4] |
|----|---------|---------|---------|---------|---------|
| 0 | 0 | ∞ | ∞ | ∞ | ∞ |
| 1 | 0 | -1 | 4 | ∞ | ∞ |
| 2 | 0 | -1 | 2 | 1 | 1 |
| 3 | 0 | -1 | 2 | -2 | 1 |
| 4 | 0 | -1 | 2 | -2 | 1 |

$n-1=4$ 轮后收敛。第 5 轮检测无变化 → 无负环。

**最终最短路径**：dist = [0, -1, 2, -2, 1]
- 0→1: 直接边权 -1
- 0→2: 0→1→2 = -1+3 = 2
- 0→3: 0→1→4→3 = -1+2-3 = -2
- 0→4: 0→1→4 = -1+2 = 1

## 时间复杂度

| 算法 | 复杂度 | 适用 |
|------|--------|------|
| Dijkstra | $O((V+E)\log V)$ | 非负权 |
| Bellman-Ford | $O(VE)$ | 任意权，检测负环 |
