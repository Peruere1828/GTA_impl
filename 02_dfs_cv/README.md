# DFSCV：Tarjan 算法找割点

## 算法简介

**割点（Cut Vertex / Articulation Point）**：删除该顶点（及其关联边）后，图的连通分量数增加。

**Tarjan 算法**使用一次 DFS 即可找出所有割点，复杂度 $O(V+E)$。

## 核心概念

- `dfn[u]`：DFS 访问顺序编号（时间戳）
- `low[u]`：顶点 $u$ 及其后代通过**后向边**能到达的最早（最小 dfn）祖先

## 算法流程

1. 从任意未访问顶点出发做 DFS
2. 进入节点 $u$ 时，`dfn[u] = low[u] = ++timer`
3. 对于 $u$ 的每个邻居 $v$：
   - 若 $v$ 未访问（树边）：
     - 递归进入 $v$
     - $low[u] = \min(low[u], low[v])$
     - 若 $low[v] \ge dfn[u]$ 且 $u$ 非根 → $u$ 是割点
   - 若 $v$ 已访问且非父节点（后向边）：
     - $low[u] = \min(low[u], dfn[v])$
4. 若 $u$ 是 DFS 根且有 ≥2 个子节点 → $u$ 是割点

### 判断条件解释

- **非根节点**：存在子节点 $v$ 满足 $low[v] \ge dfn[u]$
  - 含义：$v$ 及其后代无法通过后向边跳过 $u$ 到达 $u$ 的祖先
  - 删除 $u$ 后 $v$ 所在子树会与根分离
- **根节点**：有 ≥2 个子节点
  - 删除根后，各子树彼此不连通

## 手动模拟示例

### 图结构（蝴蝶图）

```
    0 ─ 1
    │ \ |
    2 ─ 3 ─ 4
```

（2 连接两个三角形，是割点）

### DFS 过程（从 0 出发）

| 步骤 | 进入           | dfn       | low | 子节点 v          | low[v]≥dfn[u]?                            |
| ---- | -------------- | --------- | --- | ----------------- | ----------------------------------------- |
| 1    | 0              | 1         | 1   | 1(树边)           |                                           |
| 2    | 1              | 2         | 2   | 2(树边)           |                                           |
| 3    | 2              | 3         | 3   | 3(树边)           |                                           |
| 4    | 3              | 4         | 4   | 4(树边)           |                                           |
| 5    | 4              | 5         | 5   | 2(后向边)         | low[4]=min(5,3)=3                         |
| 6    | 回溯到3        |           |     | low[3]=min(4,3)=3 | low[3]=3≥dfn[2]=3 ✓ → 2是割点 |
| 7    | 回溯到2        |           |     | low[2]=min(3,3)=3 |                                           |
| 8    | 2再看0(后向边) |           |     | low[2]=min(3,1)=1 |                                           |
| 9    | 回溯到1        |           |     | low[1]=min(2,1)=1 | 1≥2?→不是                                 |
| 10   | 0其他邻居      | 0→2已访问 |     |                   |                                           |

最终 dfn: [1,2,3,4,5], low: [1,1,1,3,3]
割点: 2（low[3]=3 ≥ dfn[2]=3）

## 时间复杂度

- DFS 一次遍历：$O(V+E)$
- 每条边处理一次

## 测试用例

1. 链 0-1-2-3：割点 {1,2}
2. 三角形：无割点
3. 蝴蝶图：割点 {2}
4. 2-连通图：无割点
5. 五边形+枝：割点 {2,5}
