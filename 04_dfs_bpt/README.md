# DFSBpt：二分图判定算法

## 算法简介

**二分图**：顶点集可分为两个不相交的子集 A 和 B，使得每条边连接 A 中一个顶点和 B 中一个顶点。

**判定定理**：图 G 是二分图 ⇔ 可以二染色 ⇔ 不含奇环。

**算法**：DFS（或 BFS）对图进行二染色，若发现相邻顶点同色则不是二分图。复杂度 $O(V+E)$。

## 算法流程

1. 初始化所有顶点颜色为未染色 (-1)
2. 对每个未染色顶点 u：
   - 将 u 染成颜色 0
   - 递归地对其邻居染相反颜色（0→1, 1→0）
   - 若某邻居已染色且颜色与当前顶点相同 → 不是二分图
3. 若所有顶点染色成功 → 是二分图

## 手动模拟示例

### 示例1：四边形 C4（是二分图）

```
0───1
|   |
3───2
```

| 步骤 | 访问 | 动作 | 染色结果 |
|------|------|------|---------|
| 1 | 0 | 染 0 | color[0]=0 |
| 2 | 0→1 | 邻居1未染色，染1 | color[1]=1 |
| 3 | 0→3 | 邻居3未染色，染1 | color[3]=1 |
| 4 | 1→2 | 邻居2未染色，染0 | color[2]=0 |
| 5 | 2→3 | 邻居3已染色=1≠0，OK | - |

结果：color=[0,1,0,1]，合法二染色 → 是二分图 ✓

### 示例2：三角形 K3（不是二分图）

```
    2
   /\
  /  \
 /    \
0──────1
```

| 步骤 | 访问 | 动作 | 染色 |
|------|------|------|------|
| 1 | 0 | 染0 | [0,-1,-1] |
| 2 | 0→1 | 染1 | [0,1,-1] |
| 3 | 0→2 | 染1 | [0,1,1] |
| 4 | 1→2 | 2已染色=1，1也是1 | 冲突！ |

→ 不是二分图 ✗（奇环不可能二染色）

## 时间复杂度

- 每个顶点和边访问一次：$O(V+E)$

## 测试用例

1. 四边形 C4：偶环，二分图 ✓
2. 三角形 K3：奇环，非二分图 ✓
3. 星形 K1,4：树，二分图 ✓
4. 树：二分图 ✓
5. 两个四边形：不连通偶环，二分图 ✓
6. 三角形+四边形：含奇环，非二分图 ✓
